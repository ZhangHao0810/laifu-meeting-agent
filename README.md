# laifu-meeting-agent
会议室预定智能体

## 核心场景：会议室预定与管理 ⭐

**业务价值**: 提升会议组织效率,智能匹配会议室资源,减少人工协调成本。

**支持的典型操作**:
- **查询类**: "现在哪个会议室空闲?" / "我明天有需要参加的会吗?"
- **预定类**: "下周三上午帮我约个产品部的季度汇报会,大概12人"
- **变更类**: "明天的季度汇报会需要提前到10点开始" / "下周四的技术月度汇报会议需要加上于强"
- **取消类**: "临时有事,明天的会不开了"

**技术现状**:
- ✅ 增删改查单个会议接口
- ✅ 查询空闲会议室接口(支持容纳人数限制)
- ✅ 按天查询会议室
- ✅ 查询会议室的与会人
- ⚠️ **缺失**: 会议室位置信息获取不到

**待解决问题**:
- 需与 IT 进一步沟通,补充会议室位置等详细信息接口
